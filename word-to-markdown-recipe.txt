## Common Patterns

### 1. Headers

**Word:**
```
Header 1
Header 1.1
Header 1.1.1
```

**Markdown:**
```markdown
# Header 1
## Header 1.1
### Header 1.1.1
```

### 2. Lists

**Word:**
```
• Item 1
    ○ Subitem 1
    ○ Subitem 2
• Item 2
```

**Markdown:**
```markdown
* Item 1
    * Subitem 1
    * Subitem 2
* Item 2
```

### 3. Tables

**Word:**
```
Column 1 | Column 2
Content  | Content
```

**Markdown:**
```markdown
| Column 1 | Column 2 |
|----------|----------|
| Content  | Content  |
```

### 4. Formatting

1. **Text Styles**
   * Bold: `**text**`
   * Italic: `*text*`
   * Both: `***text***`
   * Strike: `~~text~~`

2. **Links**
   * Inline: `[text](url)`
   * Reference: `[text][1]`
   * Footnote: `[^1]`

3. **Code**
   * Inline: `` `code` ``
   * Block:
     ````markdown
     ```language
     code block
     ```
     ````

### 5. Special Elements

1. **Images**
   ```markdown
   ![alt text](image.jpg "title")
   ```

2. **Blockquotes**
   ```markdown
   > quoted text
   > continued
   ```

3. **Horizontal Rules**
   ```markdown
   ---
   ```

4. **Footnotes**
   ```markdown
   Text[^1]
   [^1]: Footnote content
   ```
